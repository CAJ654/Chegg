# **App Name**: Chegg Tactics

## Core Features:

- Interactive Game Board: Visualize the 10x8 checkerboard with spawn zones and dynamically display minion positions and statuses.
- Turn-Based Game Engine: Implement core game logic including mana progression (1-6), minion draw mechanics, and sequential player turn management.
- Minion Action System: Enable players to perform actions: spawn minions (mana cost), move minions (free/dash/Villager specific), and attack with minions (mana cost, patterns, combo rule).
- Deck Builder: An interface for players to strategically assemble their 15-minion deck before starting a game.
- Minion Details Viewer: Provide an accessible overview of all minion statistics, movement, and attack patterns, and special abilities.
- AI Opponent: A generative AI tool that strategizes minion spawning, movement, and attacks against the human player, learning from game states to provide challenging gameplay.
- Victory Condition Detection: Automatically detect when a player's Villager is eliminated and declare the winner, ending the game.

## Style Guidelines:

- A deep, mystical purple (#751FBD) as the primary action color, suggesting arcane power and strategic depth in gameplay.
- The background color is a subtle, dark purplish-grey (#221D25), providing a brooding backdrop that enhances visual elements without distracting.
- A vibrant blue (#6675FF) is used for accent elements and interactive components, offering a clear and energetic contrast against the darker palette.
- Headlines use 'Space Grotesk' (sans-serif) for a modern, strategic feel, while 'Inter' (sans-serif) provides clear and legible body text for rules and minion descriptions.
- Abstract, minimalist icons for minion abilities and actions to maintain a clean aesthetic on the game board.
- A responsive two-column layout on wider screens, dedicating ample space to the interactive game board and a sidebar for player information and action controls.
- Subtle, fluid animations for minion movements and attacks to provide clear visual feedback without hindering gameplay speed.
